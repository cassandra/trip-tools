Itinerary

Day by day.

The Problems

  • Creating a trip itinerary is mostly manually duplicating information from other sources.
  • Updating an itinerary is not linked to other trip change activities.

The Solution

  • A itinerary that is deeply integrated with location data, planning, bookings, to-do items.
  • Planning integration to automate itinerary generation.

Existing Tools

  • TripIt
    • Good for building itinerary, but not everything in an itinerary maps to a booking.
    • The user interface is relatively poor.
    • Seems to be lightly supported since they were bought.
    • Used to have an API which would be great to itegrate with, but is no longer offered .
  • Spreadsheets
    • Great tool for creating itineries, but burden is on you and not integrated.
