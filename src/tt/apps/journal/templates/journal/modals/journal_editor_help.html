Welcome to the journal editor!

This guide covers keyboard shortcuts and editing features to help you create rich journal entries.

{% icon "pencil" %} Text Formatting
Toolbar Buttons
  • Basic: Bold, Italic, Headings (H2-H4)
  • Lists: Bulleted, Numbered, Indent/Outdent, Quotes
  • Insert: Links, Code blocks
Keyboard Shortcuts
Ctrl+B / ⌘+B Make text bold
Ctrl+I / ⌘+I Make text italic
Shift + Enter Continue indent/quote on new line

{% icon "lightbulb" %} Editing Tips
  • Auto-save: Changes save automatically after {{ TtConst.EDITOR_AUTOSAVE_INTERVAL_SECS }} seconds of inactivity
  • Image placement: Drag images into paragraphs (float-right) or between paragraphs (full-width)
  • Multi-select: Click images with Ctrl/⌘ to select multiple, or Shift for range selection
  • Reordering: Drag images within the editor to reposition them
